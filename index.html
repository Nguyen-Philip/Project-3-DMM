About Wellness In a Minute

Wellness In a Minute is a small podcast with an interactive multimedia website. It allows you to have the ability to go to different pages while also being able to interact with buttons and play an audio file of one of Wellness In a Minute's podcast. This project is coded in html without any javascript and uses css for the styling. The Website is being hosted by Github. The code itself was created on replit.com since it allows you to run the code quickly while being able to see the changes without the need of hosting it on another site.
